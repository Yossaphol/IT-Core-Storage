# Warehouse Management System (WMS)

## วิธีการ Run Server

พิมพ์คำสั่งต่อไปนี้ใน Terminal

```bash
npm start
```

---

## บัญชีสำหรับเข้าสู่ระบบ

ระบบแบ่งผู้ใช้งานออกเป็น **3 บทบาท (Roles)**

---

### 1. เจ้าหน้าที่ดูแลระบบ (System Administrator)

**Username:** `system1`
**Password:** `1234`

หน้าที่หลัก

* จัดการข้อมูลผู้ใช้งานในระบบ
* ดูแลการตั้งค่าระบบ

---

### 2. เจ้าหน้าที่คลังสินค้า (Warehouse Staff)

**Username:** `staff1`
**Password:** `1234`

หน้าที่หลัก

* รับสินค้าเข้าคลัง (IN)
* เบิกสินค้าออก (Issue)
* ปรับจำนวนสินค้า (Adjustment)
* ตรวจสอบตำแหน่งสินค้าในชั้นเก็บ

---

### 3. ผู้จัดการคลังสินค้า (Warehouse Manager)

**Username:** `admin1`
**Password:** `1234`

หน้าที่หลัก

* ตรวจสอบสถานะสินค้าในคลัง
* ตรวจสอบประวัติการเคลื่อนไหวของสินค้า
* ดูรายงานภาพรวมของคลังสินค้า

---
